# 2026-06-23-950pr-vs-950dt

- **问题**: 昇腾950PR和950DT有什么区别？
- **归档日期**: 2026-06-23
- **来源**: 基于 [[ascend-950pr-vs-950dt]]、[[ascend-950-spec-table]] 综合

## 回答摘要

**950PR 和 950DT 共架构，主要差异在定位、片上内存带宽和算力规模。**

### 定位

| | 950PR | 950DT |
|---|-------|-------|
| 场景 | 推荐、LLM Prefill、多模态推理 | 预训练、后训练、Decode+Prefill 推理 |
| 目标 | 高吞吐 + 低延迟 | 突破内存墙，覆盖训练与复杂推理 |

### 关键硬件差异

| 维度 | 950PR | 950DT |
|------|-------|-------|
| 片上内存 | 最高 128GB | 最高 144GB |
| 内存带宽 | 1.6TB/s | **4TB/s** |
| 内存模块数 | 8 个 | 4 个 |
| Cube Core（满配） | 32 | 36 |
| MXFP4 总算力（满配） | 1784 TFLOPS | 2007 TFLOPS |

### 相同点

第三代 [[davinci-core-gen3]]、[[l2-cache]]、[[unified-bus]] 2.0、[[stars2]]、[[cann]] 软件栈等架构一致。

### 选型

- 推荐 / Prefill / 多模态推理 → **950PR**
- 大模型训练 / 高带宽内存需求 → **950DT**
